# 如何从Spline获取Code Export URL

## 📋 快速步骤

### 1. 打开你的Spline项目

在Spline编辑器中打开你的珠子设计（钥匙或其他模型）

### 2. 导出Code Export

1. 点击右上角的 **Export** 按钮
2. 在弹出菜单中选择 **Code Export**
3. 选择 **React** 选项
4. 点击 **Copy URL** 或查看提供的URL

### 3. URL格式

你会得到类似这样的URL：
```
https://prod.spline.design/XXXXXXXX/scene.splinecode
```

其中 `XXXXXXXX` 是你的场景ID

### 4. 使用URL

将这个URL复制下来，准备用于更新珠子配置

---

## 🎯 测试URL是否有效

你可以在浏览器中直接访问这个URL，应该会下载一个 `.splinecode` 文件

---

## ⚠️ 注意事项

1. **公开访问**：确保你的Spline项目设置为"Public"或"Anyone with the link"
2. **URL不会过期**：这个URL是永久的（除非你删除Spline项目）
3. **更新同步**：如果你在Spline中修改了设计，需要重新导出获取新URL

---

## 📝 准备好URL后

准备好URL后，告诉我，我会帮你更新珠子配置！

示例：
- 钥匙模型的URL：`https://prod.spline.design/xxx/scene.splinecode`
- Flow模型的URL：`https://prod.spline.design/yyy/scene.splinecode`
