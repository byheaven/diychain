# 🚀 快速启动指南

## 5分钟启动 DIY Chain 编辑器

### 1️⃣ 安装依赖

```bash
npm install
```

### 2️⃣ 启动开发服务器

```bash
npm run dev
```

### 3️⃣ 打开浏览器

访问 [http://localhost:3000](http://localhost:3000)

### 4️⃣ 开始使用

1. **着陆页** → 点击"开始创作"按钮
2. **编辑器页面**：
   - 📦 **左侧**：珠子库（可筛选材质和形状）
   - 🎨 **中间**：3D 画布（拖放珠子到这里）
   - ⚙️ **右侧**：属性面板（点击珠子后调整属性）
   - 🔄 **底部**：撤销/重做控制

### 🎮 基本操作

#### 添加珠子
- 从左侧珠子库**拖动**珠子
- **释放**到中间 3D 画布

#### 编辑珠子
- **点击** 3D 场景中的珠子
- 在右侧面板调整：
  - 🔍 大小
  - 🎨 颜色
  - ✨ 金属度
  - 🌫️ 粗糙度

#### 移除珠子
- 选中珠子
- 点击右侧面板底部的"移除珠子"按钮

#### 筛选珠子
- 使用左侧面板的材质和形状按钮
- 点击"清除"重置筛选

#### 撤销/重做
- 使用底部控制栏的按钮
- 或按 Ctrl+Z / Ctrl+Y

#### 主题切换
- 点击右上角的 🌙/☀️ 图标

### 🐛 遇到问题？

1. **端口被占用**：修改端口或关闭占用 3000 端口的程序
2. **依赖安装失败**：删除 `node_modules` 和 `package-lock.json`，重新运行 `npm install`
3. **3D 场景不显示**：确保浏览器支持 WebGL
4. **样式混乱**：清除浏览器缓存并刷新

### ✨ 提示

- 使用**鼠标滚轮**缩放 3D 场景
- 使用**鼠标拖动**旋转 3D 视角
- 珠子会自动保存在浏览器本地存储中
- 支持深色和浅色主题
